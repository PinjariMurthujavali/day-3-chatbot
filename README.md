\# Day 3: AI Chatbot with Personality + Chat History Save 🎭



Advanced AI chatbot with personality (system prompts) and automatic chat history saving.



\## Features

\- System prompts for AI personality customization

\- Full conversation memory maintained

\- Automatic chat history saving to timestamped files

\- Clean conversation format in saved files

\- Exit gracefully with 'exit' command



\## Setup

```bash

python -m venv venv

venv\\Scripts\\activate

pip install groq python-dotenv

```



\## Usage

```bash

python chatbot.py

```



Chat with the AI. When you exit, chat is automatically saved to a file!
