## 📌 About This Project

Part of my **AI 90-Day Challenge** — building one real AI project every single day.

Day 3 upgrades the chatbot with **custom AI personality** through system prompts, plus **automatic conversation logging** — every chat session is saved permanently, so nothing is ever lost.

## ✨ Features

- 🎭 **Custom AI Personality** — system prompts shape how the AI responds and behaves
- 🧠 **Full Conversation Memory** — the AI remembers the entire chat context
- 💾 **Automatic Chat History Save** — every session saved to a timestamped file
- 📄 **Clean Conversation Format** — saved files are easy to read and review later
- 🚪 **Graceful Exit** — type `exit` to end the chat and trigger auto-save

## 🛠️ Tech Stack

- **Language:** Python 3.x
- **AI/LLM:** Groq API (Llama 3.3)
- **Libraries:** `python-dotenv`

## 🚀 How to Run

```bash
# Clone the repo
git clone https://github.com/PinjariMurthujavali/day-3-chatbot.git
cd day-3-chatbot

# Setup virtual environment
python -m venv venv
venv\Scripts\activate

# Install dependencies
pip install groq python-dotenv

# Add your Groq API key in a .env file
GROQ_API_KEY=your_key_here

# Run the chatbot
python chatbot.py
```

## 💬 Usage

Just start chatting naturally with the AI. Type `exit` whenever you're done — the entire conversation is automatically saved to a timestamped file, ready to review anytime.
